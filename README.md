# Dashboard

## วิธีการรัน

1. Import Project JAVA
2. รัน service-discovery-server
3. รัน service ที่ต้องการ
4. เปิด terminal
5. พิมพ์ cd dashboard -> yarn && yarn start
6. เปิด http://localhost:3000/

## Services

- Student Service
- Teacher Service
- Course Service
- Room Service
- Payment Service
